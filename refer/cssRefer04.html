CSS 색상

색상 이름으로 표현

red / green / yellow / black 등 영어로 된 이름으로 표현한다 

<style>
        .blue {
            color:blue;
        }
</style>

RGB 색상 값으로 표현

빨간색(Red), 녹색(Green), 파란색(blue)을 혼합해서 색을 표현한다 

<style>
        .blue {
            color: rgb(0,0,255);          // 빨간색(0), 녹색(0), 파란색(255)
        }

        .silver {
            color: rgb(192,192,192);
        }
</style>

16진수 색상값으로 표현

RGB 색상값을 16진수로 변환한 값, RGB의 기본색(Red, Green, Blue)은 각각 00~FF까지의 범위값을 가진다 

<style>
            .blue {
                color:#0000FF;
            }
</style>

RGBA 색상값으로 표현

RGB 색상값에 알파값을 더한 색상값이다
알파값은 색상의 투명도를 나타낸다
투명 상태값은 0.0(완전투명) ~ 1.0(불투명) 사이의 값을 가진다 

<style>
.Title{
    background-color: rgba(0,255,255,0);      // 알파값 0 추가
}
</style>
