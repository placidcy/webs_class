CSS란?

Cascading Style Sheet의 줄임말, 웹사이트의 각 요소들의 색상, 크기, 이미지크기, 위치 등 디자인 요소를 설정하는 언어

CSS 주석 표시

"/* ~ */ 기호를 사용한다

/* 주석 내용 */

CSS 선언 방법

내부 스타일

<head> 태그 안쪽의 <style> 태그 부분에 작성한다

<head>
<style>
           h1 {
                font-size: 15px;
                color: #fff;
            }
</style>
</head>

외부 스타일

다른 CSS 파일을 추가하고 해당 CSS 파일에 필요로하는 html 문서를 불러오는 방식
<link> 태그를 사용한다

<link rel="stylesheet" href="css/css.html">

인라인 스타일

html 태그에 필요한 디자인 속성을 직접 작성하는 형식, style 속성을 사용한다

<h1 style="color: blue", margin: 10px, 10px 10px, 10px>Title</h1>
