웹 표준

월드 와이드 앱 표준 개발 기구(Word Wide Web Consortium)에 의해 정해진 웹에서 표준적으로 사용되는 기술이나 규칙
웹 표준 기술로는 구조(HTML, XHTML), 표현(CSS), 동작(script)이 있다


구조(HTML)

HTML(Hyper Test Markup Language)은 국제 표준 기구가 1986년 채택한 웹에서 사용하는 표준 언어이다
태그의 사용 방법이나 오류 처리 방법이 엄격하지가 않아 웹 개발 언어로 빠르게 확산되었다


구조(XHTML)

XHTML(eXtensible Markup Language)은 1996년 월드 와이드 앱 표준 개발 기구(Word Wide Web Consortium)에서
구조화된 문서를 전송할 수 있도록 설계된 표준화된 텍스트 형식이다
HTML과는 다르게 태그의 운용 방법이나 오류 처리에 엄격하다


표현(CSS)

CSS(Cascading Style Sheet)는 웹 제작자와 사용자들의 필요에 의해 개발된 언어이다
폰트(글자), 색상, 공백, 배경 등 시각적인 요소를 자유롭게 표현할 수 있다


동작(script)

웹 페이지의 요소들을 객체화해서 동작을 제어하는 기술로 DOM, ECMA Script가 있다


DOM(document object model)

웹 페이지의 구성 체계를 나타내며 HTML 문서나 XML 문서에 접근하기 위한 인터페이스이다


EMCA script

Ecma international에서 ECMA 기술 규격에 따라 정의하는 표준 스크립트 프로그래밍 언어로
자바 스크립트를 표준화하기 위해 만들어진 언어이다



웹 접근성

장애인이나 고령자가 웹 사이트에서 제공하는 정보를 일반인과 동등하게 이용할수 있도록 보장하는 것


접근성 준수를 위한 고려 사항


1. 시각
실명, 색각 이상 등 시각 장애


2. 이동성
근육 제어 손실로 손을 쓰기 어렵거나 쓸수 없는 상태


3. 청각
영상, 음성 컨텐츠에 자막, 원고, 수화 등 대체 수단 부제로 인해 인식이 불가능한 상태


4. 인지
문제 해결과 논리 능력, 집중력, 기억력에 문제가 있는 지체, 발달 장애



웹 호환성

표준 웹 기술을 사용하여 운영체제, 브라우저 등 특정 기능에 최적화되거나 종속되지 않도록 공통적인 요소를 사용하여
웹 사이트를 제작하는 방법으로 웹 사이트를 사용할때 운영체제, 브라우저에서 동일한 결과가 나오도록 하는 개념
