속성 설명
<a> 특정 텍스트나 요소를 클릭하면 연결할 외부 파일이나 사이트 링크를 설정한다
<abbr> 약어 또는 두문자어를 나타낸다
<acronym> 성자가 단어의 약어 또는 약어를 구성하는 일련의 문자를 명확하게 나타낼 수 있다
<address> 개인 또는 조직에 대한 연락처 정보를 제공함을 나타냅니다.
<area> HTML 요소는 미리 정의된 클릭 가능한 영역이 있는 이미지 맵 내부의 영역을 정의한다
<article> 웹 문서 내용과는 다른 독립적인 내용을 포함하는 태그
<aside> 문서의 부가적인 요소의 양옆 구역을 설정한다
<audio> 문서에 사운드 콘텐츠를 포함하는 데 사용된다
<b> 특별한 중요성이 부여되지 않은 요소의 내용에 독자의 주의를 끌기 위해 사용된다
<base> 문서의 모든 상대 URL에 사용할 기본 URL을 지정한다
<bdi> 포함된 텍스트를 주변 텍스트와 분리하여 처리하도록 브라우저의 양방향 알고리즘에 지시한다
<bdo> 텍스트의 현재 방향성을 재정의하므로 내부의 텍스트가 다른 방향으로 렌더링된다
<blockquote> HTML 요소는 포함된 텍스트가 확장 인용임을 나타낸다 일반적으로 들여쓰기로 시각적으로 렌더링된다
<body> 문서의 내용을 지정한다
<br> 줄바꿈을 실행한다
<button> 사용자가 입력할 값을 받고 서버로 전달하는 태그
<canvas> 캔버스 스크립팅 API 또는 WebGL API와 함께 그래픽 및 애니메이션 그리기를 설정한다
<caption> 표의 설명 또는 제목을 표시한다
<cite> 인용된 창작물에 대한 참조를 설명하는 데 사용되며 해당 작품의 제목을 포함해야 한다, 참고문헌은 인용 메타데이터와 관련된 문맥에 적합한 관례에 따라 축약된 형태일 수 있다.
<code> 코드 영역을 구분하는 태그
<col> 표의 열(세로)부분을 설정한다
<colgroup> 표의 열을 묶어 한 그룹으로 지정한다
<data> 주어진 콘텐츠를 기계가 읽을 수 있는 번역과 연결한다
<datalist> HTML 요소에는 다른 컨트롤 내에서 선택할 수 있는 허용되거나 권장되는 옵션을 나타내는
<dd> 설명 목록(
)에서 선행 용어(
)에 대한 설명, 정의 또는 값을 제공합니다.
<del> HTML 요소는 문서에서 삭제된 텍스트 범위를 나타낸다
<details> 위젯이 "열린" 상태로 전환될 때만 정보가 표시되는 공개 위젯을 만든다
<dfn> HTML 요소는 정의 구문 또는 문장의 컨텍스트 내에서 정의되는 용어를 나타내는 데 사용된다
<dialog> 대화 상자 또는 해제 가능한 경고, 검사기 또는 하위 창과 같은 기타 대화형 구성 요소를 나타낸다
<div> 문서의 특정 요소들을 하나로 묶어 영역을 지정한다
<dl> 설명 목록을 나타낸다
<dt> 설명 또는 정의 목록에서 용어를 지정한다
<em> 글씨를 강조한다, strong보다 더 강조할때 사용한다
<embed> 문서의 지정된 지점에 외부 콘텐츠를 포함한다 이 콘텐츠는 외부 응용 프로그램이나 브라우저 플러그인과 같은 대화형 콘텐츠의 다른 소스에서 제공한다
<fieldset> 관련있는 폼의 내용을 세트로 표시한다
<figcaption> 는 상위
<figure> HTML 요소는 잠재적으로
<font> 글씨의 크기, 색상을 설정한다
<footer> 웹사이트의 하단에 문서의 부가적인 정보를 나타낸다
<form> 사용자 입력에 대한 컨트롤을 설정한다
<h1> ~ <h6> 문서 내용의 제목이나 머릿말을 나타낸다 h1 태그의 크기 설정이 가장 크고 h6 태그는 가장 작다
<head> 웹사이트의 시작 부분을 나타낸다
<header> 문서 내용의 시작 부분을 나타낸다
<hgroup> 문서 섹션에 대한 다단계 제목을 낸다

요소 집합을 그룹화한다
<hr> 가로 줄을 화면에 나타내도록 설정한다
<html> 문서의 전체 부분을 지정한다
<i> 글자를 기울여서 표시하는 태그로, italic의 약자이다
<iframeHTML> 인라인 프레임(inline frame)을 정의할 때 사용한다 인라인 프레임은 현재 HTML 문서에 다른 문서를 포함시킬 때 사용한다
<img> 사진이나 그림을 추가한다
<input> 사용자가 입력하는 구역을 설정한다
<ins> 텍스트 아래쪽에 라인(line)을 추가하여 문서에 추가된 텍스트를 표현할 때 사용한다
<kbd> 키보드(keyboard) 입력을 나타낼 때 사용한다
<keygen> 사용자가 입력한 데이터를 암호화하여 서버로 전송한다, 이때 생성된 키(key)를 가지고 서버는 해당 사용자의 인증을 수행한다
<label input> 사용자 인터페이스(UI) 요소의 라벨(label)을 정의할 때 사용한다
<legend> fieldset 태그로 묶어놓은 요소들의 제목을 표시한다,묶어놓은 요소들이 어떤 내용인지 나타낸다
<li> 목록의 데이터를 설정한다
<link> 특정 외부 파일로 이동하는 기능을 설정한다
<main> 웹 사이트 내용의 주요 내용을 구역으로 나타낸다
<map> 이미지의 특정 부분을 지정하여 링크를 걸 때 사용하는 태그이며, img 태그와 함께 사용한다.
<mark> 형광펜으로 칠한 것처럼 하이라이트된(highlighted) 텍스트를 정의할 때 사용한다
<marquee> 텍스트를 좌우상하로 움직이는 태그
<menu> 폼 컨트롤 목록이나 명령어 목록을 만들때 사용
<menuitem> 해당 페이지에서 마우스 오른쭉 눌렀을 때 나타나는 팝업 페이지에 명령이나 메뉴 사항 추가.
<meta> 웹 서버와 웹 브라우저간에 상호 교환되는 정보를 정의할때 사용
디자인에는 영향을 주지 않고 문서가 어떤 내용을 담고 있고 키워드는 무엇이고 누가 만들었는지에 대한 구체적인 문서의 정보를 나타낸다
<meter> 현재의 점수나 수치를 표시할 때 사용한다
<nav> 현재 사이트의 페이지에 연결 되는 다른 부분(네비게이션)들의 집합을 정의한다
<noframes> 프레임(frame) 기능을 사용하지 않도록 설정했거나, 프레임 기능을 지원하지 않는 브라우저를 위한 별도의 콘텐츠를 정의할 때 사용한다
<noscript> 클라이언트 사이드 스크립트(client-side scripts)를 사용하지 않도록 설정했거나, 스크립트를 지원하지 않는 브라우저를 위한 별도의 콘텐츠를 정의할 때 사용한다
<object> 다양한 멀티미디어(오디오, 비디오, 자바 애플릿, 액티브X, 플래시 등)를 웹 페이지에 삽입할 수 있으며, 웹 페이지 자체를 다른 HTML 문서에 삽입할 수 있다
<ol> 숫자로 지정된 목록을 설정한다
<옵션 메뉴를 제공하는 드롭다운 리스트(drop-down list)에서 사용되는 옵션들의 그룹을 정의할 때 사용한다>
<option select> 페이지가 로드될 때 옵션 중에서 미리 선택되어지는 옵션을 명시한다
<output> 스크립트 등에 의해 수행된 계산의 결과나 사용자의 액션에 의한 결과를 나타낼 때 사용한다
<p> 문장의 단락을 설정한다
<param object> <object> 요소의 매개변수를 정의합니다.
<picture> <img> 요소의 다중 이미지 리소스(multiple image resources)를 위한 컨테이너를 정의할 때 사용합니다.
<pre> 문장의 띄어쓰기 부분이나 여백을 그대로 나타낸다
<pregress> 진행 정도 나타내는 바를 만드는 태그
<q> 짧은 인용구(short quotation)를 정의할 때 사용한다
<rp> 루비 주석(ruby annotation)을 지원하지 않는 브라우저에서만 표시될 문자를 정의할 때 사용한다
<rt> 루비 주석(ruby annotation)에서 발음이나 설명을 나타내는 윗첨자를 정의할 때 사용한다, 동아시아 일부 지역의 언어에서 일본어의 후리가나와 같이 해당 문자의 발음이나 설명을 작은 크기의 윗첨자로 알려주는 루비 주석(ruby annotation)을 정의할 때 사용한다
<rtc> <rb> 요소가 표시하는 문자의 의미에 대한 주석을 나타냅니다.
<ruby> 동아시아 일부 지역의 언어에서 일본어의 후리가나와 같이 해당 문자의 발음이나 설명을 작은 크기의 윗첨자로 알려주는 루비 주석(ruby annotation)을 정의할 때 사용gksek
<s> 텍스트 한가운데 라인(line)을 추가하여 더 이상 정확하지 않거나 관련이 없는 텍스트를 표현할 때 사용한다
<samp> 컴퓨터 프로그램의 샘플 또는 인용 출력(output)을 나타낼 때 사용한다
<script> 자바스크립트의 코드 부분을 지정한다
<section> 특정 구역을 지정한다
<select> 다양한 선택지 중에서 선택하는 드롭박스를 만든다
<small> 덧붙이는 글이나, 저작권과 법률 표기 등의 작은 텍스트를 나타낸다
<source> 생할 미디어 파일의 URL을 명시한다
<span> 표의 특정 칸을 합치는 기능을 설정한다
<strike> 텍스트 한가운데 라인을 추가할 때 사용한다
<strong> 텍스트를 강조하는 기능
<style> CSS 코드 영역을 지정한다
<sub> 아래첨자를 넣고 싶을 때 사용합니다. subscript를 의미한다
<summary> 소에 의해 생성되는 대화형 위젯에서 기본적으로 보이는 제목을 정의할 때 사용한다
<sup> 첨자(superscript) 텍스트를 표현할 때 사용한다
<table> 표를 설정하는 코드 구역을 나타낸다
<tbody> 표의 내용 구역을 지정한다
<td> 표의 데이터를 설정한다
<template> 추가되거나 복사될 수 있는 HTML 요소들을 정의할 때 사용한다
<textarea> 사용자가 여러 줄의 텍스트를 입력할 수 있는 텍스트 입력 영역을 정의할 때 사용한다
<text-transform> 텍스트를 대문자 또는 소문자로 바꾼다
<tfoot> 표의 부가적인 내용을 가진 하단 부분을 나타낸다
<th> 표의 제목 부분을 나타낸다
<thead> 표의 제목 구역을 지정한다
<time> 일반 텍스트로 보여지는 시간, 날짜를 표현할때 사용한다
<title> 웹사이트 제목 탭의 내용을 변경한다
<tt> 텔레타이프(teletype)나 텍스트 전용 화면, 라인 프린터와 같은 고정폭 디스플레이에 표시될 고정폭 글꼴(monospace font)로 표현되는 인라인 텍스트를 생성한다
<u> 철자가 틀린 단어나 중국어의 고유 명사처럼 문체상 일반적인 텍스트와는 달라야만 하는 텍스트를 표현할 때 사용한다
<ul> 숫자가 없는 목록을나타낸다
<var> 변수(variable)를 정의할 때 사용한다
<video> 웹페이지에 영상파일을 재생할 때 사용한다
<wbr> 단어 중간에서 행바꿈될 수 있는(line-break) 위치를 정의할 때 사용한다
