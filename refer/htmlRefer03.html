시멘틱 태그

사이트의 레이아웃을 설계하기 위한 태그 / 태그에 의미를 부여해 사이트의 구조를 파악하기 위한 기능

종류

<section> <section>태그는 주제별 그룹의 콘텐츠 구역을 나타낼때 사용
<header> 사이트 내용의 제목, 머리글
<nav> 다른 페이지로의 이동을 위한 링크 공간
<aside> 왼쪽과 오른쪽 가장자리 위치의 공간을 의미하며 본문 외에 부수적인 내용을 주로 표현
<section> 주제, 카테고리별로 구역을 구분할때 사용, 같은 테마를 가진 여러개의 콘텐츠를 그룹화한다/td>
<article> 기사, 블로그 등 텍스트 위주의 페이지를 구성할때 사용
<footer> 문서의 하단에 들어가는 정보를 표현
<main> 문서의 중심 내용 또는 응용 프로그램의 중심 기능이나 관련된 콘텐츠를 나타낸다
<nav> 다른 페이지로의 이동을 위한 링크 공간
<details> 사용자가 직접 조작하여 보거나 숨길 수 있는 부가적인 세부사항을 나타낸다
<summary> details 태그 안에서 화살표 표시를 통해 생략된 내용을 보여준다
<address> 문서나 글의 저자 또는 회사와 연락할 수 있는 정보를 명시할때 사용

HTML5에서 없어진 태그

<acronym> 축약어를 표시할때 사용하는 기능
<applet> 문서에 포함되는 애플릿(웹페이지에 포함되어 작은 기능을 수행하는 프로그램)을 정의할때 사용
<basefont> HTML 문서의 기본 글꼴을 정의할때 사용한다
<big> 텍스트를 더 크게 표시할때 사용한다
<dir> 디렉토리 목록을 표시할때 사용한다
<font> 사용 글꼴을 정의할때 사용한다
<frameset> / <frame> / <noframes> 프레임 구조로 레이아웃을 작성할때 사용한다
<isindex> 검색어를 입력하는 한줄 입력 필드로 사용
<strike> 텍스트 취소선을 나타낼때 사용한다
<center> 텍스트를 가운데 정렬할때 사용한다

HTML5에서 의미가 변경된태그

<a> html4에서는 비어있는 링크를 나타낼때 href=#으로 사용
html5에서는 href 속성 없이 사용하면 자동으로 비어있는 링크로 설정된다
<address> 이름, 이메일, 전화번호 등 개인정보를 표시할때 사용되었고 html5에서는 실제 우편 주소를 나타낼때 사용된다
<b> 텍스트를 진하게 표시하고 중요하지 않지만 진하게 표시할때 사용된다
<hr> 가로줄을 표현하는 기능, html5에서는 단락 단위로 주제를 바꿀때 사용된다
<i> 텍스트를 기울어지게 표시하고 중요하지 않은 정보를 표시할때 사용된다
<menu> 실제 문서 메뉴 정보를 제공하는데 사용된다
<strong> 텍스트를 진하게 표시하는 기능뿐만 아니라 더 중요한 정보를 표시할때 사용된다
<small> 세부 주석이나 작은 인쇄 정보를 표시할때 사용된다
