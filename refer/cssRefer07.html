벤더 프리픽스

W3C의 CSS 권고안으로 확정되지 않은 속성들을 사용하거나 브라우저가 실험적으로 제공하는 기능을 쓸 때
특정 접두어를 붙여 해당 브라우저에서 인식할 수 있게 하는 기능

접두어

  • -webkit-
  • -moz-
  • -o-
  • -ms-
접두어 설명
-webkit-border-radius: 10px; 사파리, 크롬 전용
-moz-border-radius: 10px 파이어폭스 전용
-o-border-radius: 10px; 오페라 전용
-ms-border-radius: 10px; 인터넷 익스플로러 전용
