  • padding-right : 요소의 오른쪽 패딩 영역(내용(content)과 테두리(border) 사이의 간격)을 설정
  • padding-top : 요소의 위쪽 패딩 영역(내용(content)과 테두리(border) 사이의 간격)을 설정
  • page-break-after : 페이지 인쇄시 적용된 요소 뒤부터 페이지 분리에 관한 설정을 설정한다.
  • page-break-before : 페이지 인쇄시 적용된 요소 앞부터 페이지 분리에 관한 설정을 설정한다.
  • page-break-inside : p 요소 내부에서 페이지 넘김을 설정한다
  • perspective : 3차원 공간을 활성화하여 해당 요소의 자식이 해당 공간에 배치될 수 있도록 하는 데 사용된다.
  • perspective-origin : 뷰어가 3차원 공간을 바라보는 위치를 선택하는 데 사용된다.
  • pointer-events : 특정 그래픽 요소가 포인터 이벤트의 대상이 될 수 있는 조건을 제어하는데 사용된다 .
  • position : 페이지에서 요소의 위치를 설정하는 데 사용된다 .
  • quotes : 속성을 사용하여 삽입할 때 사용할 따옴표를 지정하는 데 사용된다
  • resize : 사용자가 요소의 크기를 조정할 수 있는지 여부와 가능한 경우 방향을 지정할 수 있다
  • right :위치 지정 요소의 오른쪽 오프셋을 지정하는 데 사용되는 오프셋 속성입니다 .
  • shape-image-threshold : css모양 외부에 이미지를 사용할 때 모양을 추출하는 데 사용되는 알파 채널의 임계값을 설정하는 데 사용된다.
  • shape-margin : css모양에 여백을 추가하는 데 사용된다.
  • shape-outside : float된 요소의 경계를 감싸는 방식을 제어하는 데 사용된다.
  • tab-size : 속성은 페이지에 표시되는 탭 문자의 너비를 결정하는 데 사용된다.
  • table-layout : 테이블 열, 행 및 셀의 레이아웃을 설정한다.
  • text-align : 블록 요소 내부의 인라인 요소가 정렬되는 방식을 제어하는 데 사용된다.
  • text-align-last : 강제 줄 바꿈 직전의 블록이나 줄의 마지막 줄을 정렬하는 방법을 설정하는 데 사용된다.
  • text-decoration : 텍스트 위, 텍스트 아래 또는 텍스트 위에 선 형태로 장식을 추가하거나 기존 장식을 제거하는 데 사용된다
  • text-decoration-color : 요소에 설정된 텍스트 장식(밑줄, 윗줄 및 선 통과)의 색상을 설정한다
  • text-decoration-line : 글자 라인 스타일을 설정한다.
  • text-decoration-style : text-decoration-line이 있는 요소에 설정된 선(밑줄, 윗줄 및 선 통과)의 스타일을 설정한다.
  • text-indent : 텍스트를 담고있는 블록 컨테이너에 적용되며 문단의 첫줄을 들여쓰기 할 때 사용한다
  • text-justify : 텍스트의 간격을 설정한다.
  • text-orientation : 콘텐츠 줄 내에서 문자의 방향을 설정한다.
  • text-overflow : 박스 안에 내용이 넘칠 때 텍스트를 어떻게 처리할지 지정하는 속성입니다.
  • text-rendering : 텍스트를 렌더링할 때 최적화할 항목을 선택할 수 있는 속성입니다.
  • text-shadow :텍스트에 그림자를 추가할 때 설정한다.
  • text-transform : 대문자로 또는 소문자로 바꾸는 속성입니다.
  • top : 각 옵션 속성들의 부모 태그를 기준으로 수직으로 위치를 설정한다.
  • touch-action : 어떤 요소 내에서 브라우저가 처리할 터치 액션의 목록을 설정한다.
  • transform : 요소에 회전, 크기 조절, 기울이기, 이동 효과를 설정한다.
  • transform-origin : transform 속성과 함께 사용되는 속성으로, 회전 중심(원점·기준점)을 설정한다.
  • transform-style : 자식 요소를 3D 공간에 배치하거나 2D 평면에 평평하게 배치 할지 설정한다.
  • transition : 특정 조건 하에서 변화하는 과정을 애니메이션으로 보여주고자 할 때 설정한다.
  • transition-delay : 프로퍼티가 변화한 시점과 트랜지션이 실제로 시작하는 사이에 대기하는 시간을 초 단위(s) 또는 밀리 초 단위(ms)로 설정한다.
  • transition-duration : 애니메이션 소요 시간을 설정한다.
  • transition-property : transition의 적용 여부를 설정한다.
  • transition-timing-function : 요소의 움직임 기능을 설정한다.
  • unicode-bidi : 글자의 방향 속성을 설정한다.
  • vertical-align : 텍스트의 상하 정렬 방식을 설정한다.
  • visibility : 요소를 보이지 않게 설정한다.
  • white-space : 줄바꿈, 공백, 줄 속성을 설정한다.
  • widows : 페이지, 영역 또는 열의 맨 위에 표시되어야 하는 블록 컨테이너의 최소 줄 수를 설정한다.
  • width : 요소의 가로 값을 설정한다.
  • will-change : 값이 변경될 속성에 대한 힌트를 미리 설정한다.
  • word-break : 줄바꿈 속성을 설정한다.
  • word-spacing : 단어 사이의 간격을 설정한다.
  • word-wrap :줄바꿈을 설정한다.
  • writing-mode : 텍스트의 방향(가로/세로)를 설정한다.
  • z-index : 요소의 위치가 겹칠 경우 순서를 설정한다.
