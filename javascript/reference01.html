자바스크립트 주석처리

주석은 코드를 설명하거나 추가된 다른 내용을 남길때 사용한다
코드에는 영향을 주지 않는다

한줄 주석을 남길때는 (//) 기호를 사용하고
여러 줄의 주석을 남길때는 (/* ~ */) 기호을 사용한다

1. //

// 한줄 주석

2. /* ~ */

/* 한줄
두줄
세줄
*/

코드 입력시 주의사항

1. 대문자와 소문자를 구분한다

var num;    // 두 변수의 이름은 다르다

2. 코드 한 줄을 작성 한뒤에 코드의 가장 오른쪽에 세미콜론(;)을 추가한다

const arr = new Array(10, 20, "javascript")          // 세미콜론(;)이 없어서 오류 발생
const arr = new Array(10, 20, "javascript");

3. 문자형 데이터를 작성할때 작은 따옴표(' ')나 큰 따옴표(" ")가 겹치는지 확인해야 한다

var str1 = "javascript';               // 왼쪽에는 큰 따옴표(""), 오른쪽에는 작은 따옴표(' ') -> 오류
var srt2 = 'javascript";               // 왼쪽에는 작은 따옴표(' '), 오른쪽에는 큰 따옴표("") -> 오류
