01. 1부터 100가지 출력하기

1.1 while문을 이용해서 1부터 100까지 출력하는 함수를 작성해주세요

{
    let count = 1;

    while (count <= 100) {
        console.log(count++);
    }

}

1.2 for문을 이용해서 1부터 100까지 출력하는 함수를 작성해주세요

{
    for (let count = 1; count <= 100; count++) {
        console.log(count);
    }

}

1.3 for문을 이용해서 1부터 100까지 출력하는 함수를 작성해주세요

조건 1. 짝수만 출력하세요

조건 2. 홀수만 출력하세요

{
    // 짝수
    for (let i = 1; i <= 100; i++) {
        if(i % 2 == 0){                         // %(나머지 연산자)
            console.log(i);
        }
    }

    // 홀수
    for (let i = 1; i <= 100; i++) {
        if(i % 2 != 0){
            console.log(i);
        }
    }

    // 다른 방법 1
    for (let i = 1; i <= 100; i++)
        if(i % 2 == 0) console.log(i);

    // 다른 방법 2
    for(let i = 2; i <= 100; i+=2){
        console.log(i);
    }
}

1.4 for문을 이용해서 1부터 100까지 합을 출력하시오

{
    let sum = 0;

    for(let i = 0; i <= 100; i++){
        sum = sum + i;
    }

    document.write(sum);
}
