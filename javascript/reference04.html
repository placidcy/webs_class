for()문

설정된 조건에 만족하면 반복으로 실행한다, 조건이 만족하지 않으면 실행되지 않는다

for()문 문법

const arr = [100, 200, 300, 400];

for(var i = 0; i < arr.length; i++) // i값은 0이고, arr의 배열길이(5)가 i 값보다 작으면 i값을 1씩 증가, 5번 실행된다

{
    document.write(arr[i]); // 배열 0번지부터 4번지 값을 출력(100, 200, 300, 400)
